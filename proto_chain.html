Импортируйте (с помощью директивы require или import) объект o3 под именем obj из модуля, код которого находится по адресу
 https://kodaktor.ru/j/protochain    или   https://kodaktor.ru/j/protochain2
определите цепочку прототипов этого объекта и экспортируйте результат в качестве массива имён этих прототипа.

В случае если это задание из теста, вопрос: сколько ненулевых прототипов в цепочке?
    

Вы также можете импортировать это задание из npm-пакета.

Создайте проект и выполните yarn add goss_proto

I. В REPL или в старом стиле
const { o3 } = require('goss_proto');

II. В файле-модуле (не забыв о "type":"module" в package.json)
import o3 from 'goss_proto';


   
{
 const person = { fname: 'Elias' };
 Reflect.setPrototypeOf(person, { lname: 'Goss' });
 Out.slog(person);       // у объекта одно собственное свойство
 Out.slog(person.lname); // из прототипа
}
{
import protochain from 'goss_proto'
var a=protochain.o3
var array = []
while (Object.getPrototypeOf(a) != null) {
    const proto = Object.getPrototypeOf(a) //берем прототип у a
    console.log(proto.name)
    a = proto
    array.push(proto.name)
}
// В цепочке 3 ненулевых прототипа
console.log(array)
export {array}

}
